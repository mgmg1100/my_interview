# Cursor Skills

Project skills for interview practice. Cursor discovers these automatically when this repository is open.

- `interview-learning` — one new interview item per session
- `interview-recap` — delayed recaps, daily retention tests, and monthly exams

For Codex, symlink these directories into `~/.codex/skills/` (see [codex-skills/README.md](../../codex-skills/README.md)).
