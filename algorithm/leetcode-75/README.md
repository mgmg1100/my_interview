# LeetCode 75

Use this folder for one note per problem or reusable pattern from the LeetCode 75 track.

## Files per problem

- `<kebab-case-problem-name>.md` — pattern, invariant, complexity, edge cases, think-out-loud phrases, follow-ups, mistakes
- `<kebab-case-problem-name>.py` — practice file with solution stub and runnable test cases (created during interview practice)

Run practice tests:

```bash
python algorithm/leetcode-75/<problem-name>.py
```

Each note should include:

- Problem intuition
- Brute force vs optimal approach
- Core invariant
- Complexity
- Think-out-loud phrases
- Edge cases
- Follow-up questions and answers
- Practice mistakes, when they happen
